## How to install and run project

First you should have already installed Nodes.js and npm (Node Package Manager). Then after that you could do two options

### Zip 

Copy the project 

```bash
  code ./<name-who-you-setted>
  git clone https://github.com/Al-Huann-dev/interactive-blobbing-particles.git .
  npm install
  npm run dev
```

### Directly with Git

```bash
  mkdir <name-who-you-want>
  code ./<name-who-you-setted>
  git clone https://github.com/Al-Huann-dev/interactive-blobbing-particles.git .
  npm install
  npm run dev
```
